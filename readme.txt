Add a pin-it button on every image of every WordPress gallery! Let readers to easily pin your picture!

== Description ==

Pinterest for Galleries adds a pin-it button under each thumbnail of WordPress galleries
(the ones added to posts with the native WordPress gallery feature).

It does not work with NextGen Gallery and may not work with customized galleries (it
depends on themes and/or other plugins).

Discussion forum: http://www.satollo.net/forums/forum/pinterest-for-galleries-plugin

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin
